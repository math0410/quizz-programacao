🧠 Quiz de Programação

Um projeto simples de quiz interativo sobre programação, desenvolvido utilizando apenas HTML e CSS. Ideal para praticar conceitos básicos de front-end e lógica de apresentação.

🚀 Demonstração

Você pode visualizar o projeto abrindo o arquivo index.html em seu navegador.

📌 Funcionalidades
Exibição de perguntas de múltipla escolha
Feedback visual para respostas corretas e incorretas
Layout responsivo
Interface simples e intuitiva
Sem uso de JavaScript (apenas HTML e CSS)
🛠️ Tecnologias Utilizadas
HTML5
CSS3

🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de:

Praticar estruturação de páginas com HTML
Estilização com CSS
Criar interações básicas sem JavaScript
Simular um sistema de quiz simples
